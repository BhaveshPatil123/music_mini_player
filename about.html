🎧 About This Project

Spotify Mini Player is a sleek, web-based audio player built using HTML, CSS, and JavaScript. It mimics core functionality of Spotify to provide a practical and fun learning experience in frontend development.

✨ Key Features:

  • ▶️ Play / Pause songs
  • ⏮️ Previous / Next song support
  • 🎶 Real-time display of the playing song
  • 📊 Live progress bar with time updates
  • 🔁 Automatically play next track

This project is built for educational purposes, to explore how audio, DOM, and event handling work together in a real web application.
